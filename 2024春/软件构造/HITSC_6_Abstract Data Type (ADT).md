# Abstraction and User-Defined Types
📕数据抽象：指由一组操作所刻画的数据类型
传统的类型关注数据的具体表示，而抽象类型强调“作用于数据上的操作”，无需关注数据如何存储，而是设计并使用操作
即**ADT是由操作定义的，与其内部如何实现无关！**
比如，定义一个抽象的Bool类，Bool可以由很多的东西实现，但它的操作决定了它的性质
