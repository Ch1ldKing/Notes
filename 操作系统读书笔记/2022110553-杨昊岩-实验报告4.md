# 实验内容
### 增加 `iam()`和`whoami()`
1. 首先要在`system_call.s`中增加系统调用总数，从72改为74
```c
nr_system_calls = 74
```
2. 需要定义这两个调用，在`unistd.h`头文件中
```c
#define __NR_iam    72
#define __NR_whoami 73
```
3. 将这两个系统调用加入到调用表中`linux/sys.h`，注意此处的行号需要和刚刚定义的一致
```C
72 extern int sys_iam();
73 extern int sys_whoami();
```
